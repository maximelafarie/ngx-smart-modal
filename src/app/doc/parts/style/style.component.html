How to make ngx-smart-modal fit your needs
This documentation section is for ngx-smart-modal >= 6.0.0 only! For previous versions, you only can override default CSS properties.
ngx-smart-modal provides built-in SCSS variables that you can override easily like it (assuming you imported ngx-smart-modal.scss as explained above): 
{{ CODES?.SCSS_IMPORT }}
Please note that variables needs to be overridden before @import!
