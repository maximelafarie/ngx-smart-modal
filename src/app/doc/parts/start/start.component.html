Getting started
How to use ngx-smart-modal in your projects
Installation
To use ngx-smart-modal in your project install it via npm: 
npm i ngx-smart-modal --save
or with yarn: 
yarn add ngx-smart-modal
or even with UNPKG. E.g.: if you need to use ngx-smart-modal with direct file imports, with project that use SystemJS or in live editors like JsFiddle, Codepen and so on: 
https://unpkg.com/ngx-smart-modal@{{ libVersion }}/bundles/ngx-smart-modal.umd.js
